## 📌 Sobre o Projeto

O **Network Analyser** é uma ferramenta em Python para **diagnóstico de rede**, desenvolvida para facilitar testes técnicos de forma rápida para análise e testes básicos de rede via terminal, com interface interativa e visual estilizado.

---

## ⚙️ Funcionalidades

✔ Exibição de informações de rede (IP)
✔ Listagem da tabela ARP
✔ Tracert para qualquer IP ou domínio
✔ Teste de loop automático
✔ Teste de ping com suporte a IPv4 e IPv6
✔ Menu interativo com navegação simples
✔ Animações e feedback visual no terminal

---

## 🎮 Como Usar

Ao iniciar o script, você verá um menu como este:

```
1 - Dados da rede
2 - Lista de IP's
3 - Tracert
4 - Teste de Loop
5 - Teste de Ping
6 - Instruções Técnicas
0 - Sair
```

Basta digitar o número da opção desejada e seguir as instruções na tela.

---

## 📊 Exemplo de Uso

```
Escolha um host: 1
Escolha a versão do IP: 1

Ping para www.google.com (IPv4)
```
