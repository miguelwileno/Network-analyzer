## 📌 Sobre o Projeto

O **Network Analyser** é uma ferramenta em Python para **diagnóstico de rede**, desenvolvida para facilitar testes técnicos de forma rápida e interativa diretamente no terminal.

Com uma interface estilizada e animações, ele permite executar comandos essenciais de rede sem precisar digitar manualmente no sistema.

---

## ⚙️ Funcionalidades

✔ Exibição de informações de rede (IP)
✔ Listagem da tabela ARP
✔ Tracert para qualquer IP ou domínio
✔ Teste de loopback automático
✔ Teste de ping com suporte a IPv4 e IPv6
✔ Menu interativo com navegação simples
✔ Animações e feedback visual no terminal

---

## 🎮 Como Usar

Ao iniciar o script, você verá um menu como este:

```
1 - Dados da rede
2 - Lista de IP's
3 - Tracert
4 - Teste de Loopback
5 - Teste de Ping
6 - Instruções Técnicas
0 - Sair

```

Basta digitar o número da opção desejada e seguir as instruções na tela.

---

## 📊 Exemplo de Uso

```
Escolha um host: 1
Escolha a versão do IP: 1

Ping para www.google.com (IPv4)
```
